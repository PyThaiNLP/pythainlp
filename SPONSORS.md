# Sponsors

Current PyThaiNLP remains an independent community with no affiliation
with any organization.
We don't have a Patreon or GitHub sponsor for the organization.

However, some development leads have GitHub sponsors.
You can sponsor them to help make an open-source project.

- [bact](https://github.com/sponsors/bact)
- [wannaphong](https://github.com/sponsors/wannaphong)

And you can sponsor by [contributing to PyThaiNLP](https://github.com/PyThaiNLP/pythainlp/blob/dev/CONTRIBUTING.md)!
